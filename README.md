# AI Text To Speech

A multilingual AI Text-To-Speech web application built using:

- FastAPI
- Coqui XTTS v2
- HTML/CSS/JavaScript
- Python

## Features

- English voice generation
- Hindi voice generation
- AI-powered speech synthesis
- Audio download support
- Modern web interface

## Technologies

- FastAPI
- XTTS v2
- Python
- JavaScript

## Run Locally

```bash
pip install -r requirements.txt
uvicorn main:app --reload
